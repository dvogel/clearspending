Completeness

When the government publishes spending data they are required to fill out data fields in a particular manner. These requirements are spelled out in the data dictionary. If you want to analyze or audit this spending you will need to know basic information like which agency spent the money, which program it was spent on, where the money was spent, and who received the money. Unfortunately these basic pieces of information are missing from much of the spending data.

While most of the data meets basic completeness criteria, there are areas that are particularly egregious. In the diagram below you can see these data quality hot spots. Each column in the diagram represents a column in the data released by USASpending.gov. You can move your cursor over each cell for more details.
